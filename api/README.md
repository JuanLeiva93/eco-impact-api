# EcoImpact API
Calculadora de huella de carbono digital basada en estándares de la industria.

## Instalación
1. Clona el repositorio.
2. Instala dependencias: `pip install fastapi uvicorn python-dotenv`
3. Crea un archivo `.env` en la raíz con: `API_KEY=tu_clave_aqui`
4. Ejecuta: `uvicorn main:app --reload`
